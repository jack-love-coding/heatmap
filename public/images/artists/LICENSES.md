# Artist Dossier Images

Artist portraits in this folder are local 512x512 WebP assets for the atlas UI. Four files were downloaded from Wikimedia Commons before the source host started rate-limiting this run; the remaining files use the planned generated-image fallback and are explicitly marked as generated assets.

`generated-archive-musician.png` is the older generated archival-style placeholder. It remains in the folder as a fallback asset, but current artist dossiers point to individual portraits.

## Generated Fallback Images

Generated fallback images use this prompt pattern with each artist name and role substituted: black-and-white realistic archival-style head-and-shoulders portrait for a WWII music atlas UI, vintage studio photograph look, subtle film grain, clear face, neutral backdrop, no text, no watermark, no logos, not a literal historical photograph.
