# Artist Dossier Images

`generated-archive-musician.png` is a generated archival-style placeholder created for this project with OpenAI image generation.

- It is not a historical photograph.
- It does not depict a specific real person.
- It is used only when a verified public-domain or clearly licensed portrait has not yet been added.

When replacing a placeholder with public material, add the image under this directory and update the matching artist `portrait` metadata with source URL, credit, license label, license URL where available, and `generated: false`.
